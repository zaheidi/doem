一、背景
项目简介 (Project Overview)
- 项目名称：“内推小程序”。
- 核心目标：为校友提供职位发布平台，通过老师审核机制确保信息真实，并利用 AI 技术辅助学生进行职位匹配评估。
- 适用人群：校友（发布者）、老师（审核者）、学生（求职者）分端。
技术架构 (Technical Stack)
- 前端框架：微信小程序原生框架、TDesign 零售小程序组件库（tdesign-miniprogram）。
- 后端服务：微信小程序云开发（Cloud Development）。
  - 数据库：云数据库（NoSQL）。
  - 云函数：处理审核逻辑、AI 接口调用、权限校验。
- 外部 API：例如腾讯云 AI 或 DeepSeek 等大模型接口（用于 AI 匹配度评估）。
